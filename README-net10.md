## 路线说明

- 这是 .NET 10 独立路线，和默认 .NET 8 兼容正式版分开。
- 此包只包含程序文件、说明和快捷方式脚本，不包含源代码。
- 普通用户默认建议使用 net8 正式版；需要 net10 时才使用此路线。
- 后续 net10 可以单独更新 ersion-net10.json，不会影响默认 ersion.json。

解压后可以运行“程序文件\\WhatsAppRealtimeTranslator.exe”，也可以双击“创建桌面快捷方式.cmd”后从桌面快捷方式启动。
