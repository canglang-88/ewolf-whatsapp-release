# 【亿泰坊】通讯翻译中枢 下载指引

当前最新版本：5.0.7

## 推荐发给别人：一键覆盖更新包

适合：对方电脑已经有旧版中枢，但收不到自动更新，或者不会手动覆盖文件。

点击下载：[yitaifang-suite-one-click-update-v5.0.7.zip](https://github.com/canglang-88/ewolf-whatsapp-release/releases/download/v5.0.7/yitaifang-suite-one-click-update-v5.0.7.zip)

使用方式：解压后双击 一键覆盖更新到5.0.7.exe。它会读取桌面的“【亿泰坊】通讯翻译中枢”快捷方式，自动覆盖那个快捷方式指向的程序目录。

不会覆盖：API Key、代理、账号、登录状态、聊天记录、翻译缓存。

## 自动更新

适合：已经带自动更新能力的版本。

- 5.0.1 及以上版本会优先下载小补丁包：[yitaifang-communication-suite-patch-v5.0.7-net8.zip](https://github.com/canglang-88/ewolf-whatsapp-release/releases/download/v5.0.7/yitaifang-communication-suite-patch-v5.0.7-net8.zip)
- 关闭软件后重新打开中枢即可检测到 5.0.7。
- 更新只替换程序文件，不覆盖数据。

## 新电脑或没有旧版：完整包

点击下载：[yitaifang-communication-suite-v5.0.7-net8.zip](https://github.com/canglang-88/ewolf-whatsapp-release/releases/download/v5.0.7/yitaifang-communication-suite-v5.0.7-net8.zip)

完整包适合新装；如果对方已经有旧版，优先发“一键覆盖更新包”。

## 不包含内容

- OpenAI API Key
- WhatsApp / Google / Gmail 登录数据
- 聊天记录
- 翻译缓存
- 日志和本机配置备份
- Signal 模块
