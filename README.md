## 说明

以后每次发布都保留三种包：

- 自动更新补丁包：给已经有更新能力的中枢自动下载。
- 一键覆盖更新包：给已有旧版但收不到更新的人，自动读取桌面入口并覆盖程序文件。
- 完整包：给新电脑或完整重装使用。

覆盖和自动更新都只替换程序文件，不覆盖 API、代理、账号、登录状态、聊天记录、翻译缓存。
