# 【亿泰坊】WhatsApp专用 下载指引

当前最新版本：2.1.3

## 直接下载

点击下载：[yitaifang-whatsapp-v2.1.3.zip](https://github.com/canglang-88/ewolf-whatsapp-release/releases/download/v2.1.3/yitaifang-whatsapp-v2.1.3.zip)

## 旧版本自动更新

如果对方电脑可以正常访问 GitHub，并且已经安装旧版本：关闭软件后重新打开，旧版本会检测到 2.1.3，弹窗提示更新，点击确认即可自动下载并替换程序文件。

如果对方电脑访问不了 GitHub：可能不会正常弹出更新提示，需要把上面的 zip 下载链接发给对方手动下载、解压后使用。

## 本次更新

- 多账号启动改为分批加载：第一个账号立即加载，后续账号每个间隔 2-4 秒，降低低配置电脑启动闪烁和卡顿。
- 新增“炫彩效果”开关，默认关闭，低配电脑更稳定；需要效果时可手动打开。
- 左侧账号栏改为自绘可拖动细滚动条，避免系统原生滚动条一直显示或闪动。
- 保留长消息“查看更多”完整翻译缓存、译文复制表情保留、发送防重复等优化。
- 继续保留“创建桌面快捷方式.cmd”。

## 使用提示

解压后可以运行“程序文件\WhatsAppRealtimeTranslator.exe”，也可以双击“创建桌面快捷方式.cmd”后从桌面快捷方式启动。
