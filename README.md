## 旧版本自动更新

如果电脑可以正常访问 GitHub，已安装旧版本的用户完全退出软件后重新打开，会检测到 2.0.12，并弹出更新提示，点击确认即可自动下载并替换程序文件。

如果对方电脑访问不了 GitHub，软件不会正常弹出更新提示。请把上面的压缩包链接或下载好的 zip 发给对方手动安装。

## 本次更新

- 优化翻译失败处理：失败后只自动重试一次。
- 若仍失败，会停止自动循环，并显示点击重试，避免一直在“正在翻译/翻译失败”之间重复刷屏和消耗请求。
- 保留左侧账号栏动态缩放，账号多出现滚动条时避免遮挡。
