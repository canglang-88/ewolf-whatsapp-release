## 直接下载

点击下载：[yitaifang-whatsapp-v2.1.1.zip](https://github.com/canglang-88/ewolf-whatsapp-release/releases/download/v2.1.1/yitaifang-whatsapp-v2.1.1.zip)

## 旧版本自动更新

如果对方电脑可以正常访问 GitHub，并且已经安装旧版本：关闭软件后重新打开，旧版本会检测到 2.1.1，弹窗提示更新，点击确认即可自动下载并替换程序文件。

如果对方电脑访问不了 GitHub：可能不会正常弹出更新提示，需要把上面的 zip 下载链接发给对方手动下载、解压后使用。
