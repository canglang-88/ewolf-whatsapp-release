## 旧版本自动更新

如果电脑可以正常访问 GitHub，已安装旧版本的用户重新打开软件后会检测到 2.0.10，并弹出更新提示，点击确认即可自动下载并替换程序文件。

如果对方电脑访问不了 GitHub，软件不会正常弹出更新提示。请把上面的压缩包链接或下载好的 zip 发给对方手动安装。
