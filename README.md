# 【亿泰坊】WhatsApp专用 下载指引

当前最新版本：2.1.4

## 直接下载

点击下载：[yitaifang-whatsapp-v2.1.4.zip](https://github.com/canglang-88/ewolf-whatsapp-release/releases/download/v2.1.4/yitaifang-whatsapp-v2.1.4.zip)

## 旧版本自动更新

如果对方电脑可以正常访问 GitHub，并且已经安装旧版本：关闭软件后重新打开，旧版本会检测到 2.1.4，弹窗提示更新，点击确认即可自动下载并替换程序文件。

如果对方电脑访问不了 GitHub：可能不会正常弹出更新提示，需要把上面的 zip 下载好，发给对方手动解压使用。

## 本次更新

- 修复可能弹出 .NET 安装/更新提示的问题，发布包改为 win-x64 自包含版。
- 接收长消息翻译上限从 4000 字符提高到 12000 字符。
- OpenAI 文本翻译输出上限提高到 6000 tokens，改善长文翻译不显示或不完整。

## 使用提示

解压后可以运行“程序文件\\WhatsAppRealtimeTranslator.exe”，也可以双击“创建桌面快捷方式.cmd”后从桌面快捷方式启动。
