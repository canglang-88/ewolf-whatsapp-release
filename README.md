# 【亿泰坊】WhatsApp专用 下载指引

当前最新版本：2.0.15

## 直接下载

点击下载：[yitaifang-whatsapp-v2.0.15.zip](https://github.com/canglang-88/ewolf-whatsapp-release/releases/download/v2.0.15/yitaifang-whatsapp-v2.0.15.zip)

## 旧版本自动更新

如果对方电脑可以正常访问 GitHub，并且已经安装旧版本：关闭软件后重新打开，旧版本会检测到 2.0.15，弹窗提示更新，点击确认即可自动下载并替换程序文件。

如果对方电脑访问不了 GitHub：软件不会正常弹出更新提示，需要把上面的 zip 下载链接或压缩包发给对方手动更新。

## 本次更新

- 当 WebView 跑到 WhatsApp 官网页面或错误页时，自动拉回 https://web.whatsapp.com/。
- 避免部分电脑打开后停在 WhatsApp 官网“找不到内容”页面。

解压后运行 程序文件\WhatsAppRealtimeTranslator.exe。
