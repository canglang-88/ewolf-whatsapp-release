# 【亿泰坊】WhatsApp专用 下载指引

当前最新版本：2.1.6

## 直接下载

点击下载：[yitaifang-whatsapp-v2.1.6.zip](https://github.com/canglang-88/ewolf-whatsapp-release/releases/download/v2.1.6/yitaifang-whatsapp-v2.1.6.zip)

## 旧版本自动更新

如果对方电脑可以正常访问 GitHub，并且已经安装旧版本：关闭软件后重新打开，旧版本会检测到 2.1.6，弹窗提示更新，点击确认即可自动下载并替换程序文件。

如果对方电脑访问不了 GitHub：可能不会正常弹出更新提示，需要把上面的 zip 下载后发给对方手动解压使用。

## 本次更新

- 单独 WhatsApp 专用版，不是通讯翻译系统组合版。
- 改为 .NET 8 win-x64 自包含兼容版，减少部分 Windows 10 电脑点 exe 没反应或 .NET 提示问题。
- 保留长消息翻译 12000 字符、OpenAI 输出 6000 tokens。
- 包含快捷方式创建修复、账号栏显示、头像昵称识别、滚动条和低配启动间隔等稳定性优化。

## 使用提示

解压后运行 程序文件\\WhatsAppRealtimeTranslator.exe，也可以双击 创建桌面快捷方式.cmd 后从桌面快捷方式启动。
